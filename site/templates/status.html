Project Status

As of February 13th, 2012 we have data for 47 states, the District of Columbia, and Puerto Rico.

States are denoted as either "ready" or "experimental" depending on how reliable the source data is and how thoroughly we've vetted and cleaned up the scraped data. We do our best to provide high quality data for all states, but you are more likely to find incomplete data where we haven't been able to identify a source or done manual reconciliation of ids when using experimental states.

If you are curious about the specific status of state data check below for any state specific issues.

If you encounter issues using data from any state please let us know via the Open State Project Issue Tracker or mention it on the Open States google group.

State Status

Ready States

Ready states (green on the above map) are states that are regarded as the highest quality, they have been stable for a long time and have gone through an additional QA procedure. Currently this includes Alaska, Arizona, Arkansas, California, Idaho, Kentucky, Louisiana, Maryland, Minnesota, Mississippi, Nevada, New Jersey, North Carolina, Ohio, Oregon, Pennsylvania, Texas, Utah, Vermont, Washington DC, and Wisconsin.

Experimental States

Experimental states (in blue) are those that we support but have not yet promoted to ready status. Alabama, Colorado, Connecticut, Delaware, Florida, Georgia, Hawaii, Illinois, Indiana, Iowa, Kansas, Maine, Massachusetts, Michigan, Nebraska, New Hampshire, New Mexico, New York, North Dakota, Oklahoma, Puerto Rico, Rhode Island, South Carolina, South Dakota, Virginia, Washington, West Virginia, and Wyoming fall in this category.

Unsupported States

The remaining states (gray on the map) are not yet in our API. If you are interested in contributing visit the Contributor's Guide that will guide you through obtaining the source and speaking with project developers that can help you determine what work needs to be done for the state of your choosing.

State-specific Issues

Colorado
  • Colorado does not provide a reliable source for roll call votes but does have easily parsable committee votes. We'll continue to investigate options to provide Colorado's roll call votes in the future.

Hawaii
  • Hawaii does not provide a reliable source for roll call votes but does have easily parsable committee votes. We provide all votes that are available via http://www.capitol.hawaii.gov/.

Indiana
  • Indiana only makes their votes available in scanned PDFs that currently prevent us from capturing legislator roll calls.

Iowa
  • Iowa does not provide roll call votes in a format that we can parse with any reliability, so at the moment we do not provide any roll call votes for Iowa.

Kentucky
  • Kentucky only makes their votes available in scanned PDFs that currently prevent us from capturing legislator roll calls.

Massachusetts
  • Massachusetts does not provide a source for individual roll call votes, unfortunately this means we are unable to provide this data as well.
  • It should also be noted that districts in Mass. do not obey a typical alphanumeric format, and have somewhat unreliable names. An attempt has been made to clean these up into a standard format but it is likely to be a source of some confusion when using their data.

Minnesota
  • Minnesota does not provide legislator roll calls for votes taken in the Senate so we are lacking detailed votes for the MN Senate.

Nebraska
  • Nebraska does not provide roll call votes in a format that we can parse with any reliability, so at the moment we do not provide any roll call votes for Nebraska.

North Dakota
  • North Dakota currently does not seem to provide detailed roll call votes so we are lacking this information.
